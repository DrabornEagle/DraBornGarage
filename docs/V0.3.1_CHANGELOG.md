# DraBornGarage v0.3.1 — Değişiklik Kaydı

## Sürüm bilgisi

- Yeni sürüm: `v0.3.1`
- Önceki sürüm: `v0.3.0`
- Önceki sürüm yedeği: `backup/v0.3.0-before-v0.3.1`
- Veritabanı rollback: `supabase/rollbacks/rollback_v0_3_1_to_v0_3_0.sql`

## Eklenen standartlar

- Her tamamlanan değişiklikten sonra sürüm numarası artırılır.
- Aynı yol haritası aşamasındaki değişiklikler patch numarasını artırır.
- `v0.3.1` sonrasında yapılacak değişiklik `v0.3.2` olur.
- Yol haritasında v0.4 aşamasına geçildiğinde sürüm `v0.4.0` olur.
- Yeni sürüme başlamadan önce mevcut tam sürüm için GitHub yedeği oluşturulur.
- Kurulum komutu kurulan ve yedeklenen sürümü ekranda gösterir.
- Geri alma komutu geri dönülen ve geri alma öncesinde korunan sürümü gösterir.
- Veritabanı değişikliği olmayan sürümlerde no-op rollback dosyası oluşturulur.
