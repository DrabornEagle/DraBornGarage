## Performans doğrulaması

Supabase performans danışmanı çalıştırıldı. v0.4 tablolarındaki eksik foreign-key indeksleri tamamlandı:

- `idx_work_order_events_workshop`
- `idx_work_order_extra_request_events_workshop`
- `idx_work_order_notes_workshop`

Kalan bildirimler düşük kullanımlı geliştirme veritabanındaki `unused_index` bilgi mesajlarıdır; eksik foreign-key indeksi kalmamıştır.
