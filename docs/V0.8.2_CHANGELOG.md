# DraBornGarage v0.8.2 — Ayrı Admin Paneli

## Neden gerekliydi?

Admin hesabı uygulamada doğru şekilde tanınıyordu; ancak alt menüde ayrı bir **Admin** sekmesi yoktu. Admin yetkileri `Merkez` ekranındaki `Ekip` bölümünün içine gizlenmişti ve ekran varsayılan olarak Platform bölümünü açıyordu. Bu nedenle Admin paneli yokmuş gibi görünüyordu.

## Düzeltilen yapı

- Admin hesabında alt menüde artık açıkça **Admin** sekmesi gösterilir.
- Admin sekmesinin simgesi `shield-checkmark` olarak değiştirildi.
- Admin sekmesi açıldığında doğrudan **Admin Yönetim Merkezi** görüntülenir.
- İşletme Sahibi hesaplarında mevcut **Merkez** yapısı korunur.
- Usta hesaplarında mevcut **Rapor** yapısı korunur.
- Çırak hesaplarında Admin sekmesi hiçbir zaman gösterilmez.

## Yeni Admin Yönetim Merkezi

Admin panelinde üç ana bölüm vardır:

1. **Admin**
   - Tüm işletmeleri görüntüleme
   - Yeni işletme oluşturma
   - İşletme seçme
   - İşletmeyi aktif/pasif yapma
   - Seçili işletme bilgilerini düzenleme
   - Usta daveti oluşturma
   - Çırak daveti oluşturma
   - İşletme Sahibi + Usta daveti oluşturma
   - Sadece İşletme Sahibi daveti oluşturma
   - Seçili işletmenin personelini görüntüleme
   - Usta/Çırak rol değişikliği
   - Personeli aktif/pasif yapma

2. **Raporlar**
   - Seçili işletmenin toplamları
   - Usta bazlı kayıtlı işlem tutarları
   - Tahsilatlar ve servis hareketleri

3. **Platform**
   - İşletme bazlı platform hizmet bedeli
   - Dönem borcu ve devreden borç
   - Ödeme bildirimleri
   - Dekont ve Admin onayı
