# DraBornGarage v1.0.0 RC — Release APK Test Planı

Bu plan, ilk GitHub Release APK çıktısının Google Play AAB aşamasından önce fiziksel Android cihazlarda doğrulanması içindir.

- [ ] `npm ci` başarılı
- [ ] Expo Doctor başarılı
- [ ] TypeScript başarılı
- [ ] Expo public config oluşturuldu
- [ ] Temiz Android prebuild başarılı
- [ ] Android `lintRelease` başarılı
- [ ] Android `assembleRelease` başarılı
- [ ] APK imzası doğrulandı
- [ ] APK SHA-256 checksum üretildi
- [ ] GitHub artifact yüklendi

## Bilinen yayın ayrımı

Bu APK ilk fiziksel cihaz test adaydır. GitHub Actions içinde üretilen test imzası Google Play üretim imzası değildir. Testler kapatıldıktan sonra:

1. Kalıcı Android upload key güvenli biçimde oluşturulacak.
2. GitHub Secrets veya EAS Credentials içinde saklanacak.
3. Production AAB üretilecek.
4. Play App Signing etkinleştirilecek.
5. AAB kapalı teste yüklenecek.
